## Summary Table

| Aspect              | Supervised Predictor       | RL Agent                        |
|----------------------|----------------------------|---------------------------------|
| Data access          | Full dataset, all at once  | One observation at a time       |
| Labels / targets     | Known (future prices)      | None вҖ” only reward signal       |
| Environment needed?  | No                         | Yes (black box simulator)       |
| Feedback             | Immediate (loss per sample)| Delayed (reward after actions)  |
| Training signal      | Gradient of loss function  | Policy gradient / value estimate|
| Exploration          | N/A                        | Must explore to discover reward |
| Stationarity         | Assumes i.i.d. samples     | Sequential, non-stationary      |
| What it learns       | "What will happen"         | "What to do"                    |
